# Asset Manager — บริหารสินทรัพย์ส่วนตัวสำหรับคนไทย

เครื่องมือวางแผนการเงินส่วนตัว 4 ด้าน ในไฟล์ HTML เดียว ทำงานในเบราว์เซอร์ — **ข้อมูลเก็บในเครื่องคุณเท่านั้น**

## ฟีเจอร์
- 📊 **สรุป** — มูลค่าสินทรัพย์สุทธิ (net worth) + ความคืบหน้า passive income ต่อเป้าหมาย
- 📈 **การลงทุน** — พอร์ตหุ้น + ตั้งเป้าหมายเกษียณ → คำนวณผลตอบแทนที่ต้องการ (รวมเงินเติม) + projection + สัดส่วนพอร์ตเทียบกรอบ
- 🏠 **อสังหาริมทรัพย์** — มูลค่า / จำนอง / ค่าเช่า / equity / LTV
- 🛡️ **ประกัน** — ทุนคุ้มครอง / เบี้ย / มูลค่าเงินสด / ลดหย่อนภาษี / ครบกำหนด
- 🤖 **AI prompt** — สร้างข้อความให้ AI ช่วยประเมินพอร์ต หรือหาหุ้นตามกรอบให้คะแนน
- 💾 **Export / Import JSON** — ย้ายข้อมูลข้ามเครื่อง

## ความเป็นส่วนตัว
ไม่มี backend ไม่มีการส่งข้อมูลออกจากเครื่อง — ทุกอย่างเก็บใน `localStorage` ของเบราว์เซอร์คุณ
ย้ายเครื่อง: กด **Export** ได้ไฟล์ `.json` → ส่งไปอีกเครื่อง → กด **Import**
*ไม่มี auto-sync* (ตามตั้งใจ เพื่อให้ข้อมูลอยู่ในมือคุณ 100%)

## เปิดใช้งาน
- ออนไลน์: เปิด URL ที่ deploy ไว้
- ออฟไลน์: ดาวน์โหลด `index.html` แล้วดับเบิลคลิก — ทำงานได้โดยไม่ต้องต่อเน็ต

## เทคนิค
ไฟล์ HTML เดียว · vanilla JS · ไม่มี dependency · ไม่มี build step · ไม่มี network request
ทดสอบสูตรคำนวณ: เปิด `?selftest` ต่อท้าย URL
